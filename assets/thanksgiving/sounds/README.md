# Thanksgiving Sound Effects

## Required Assets:

### Turkey Gobble Sound
- **File:** `turkey-gobble.mp3`
- **Duration:** 1-3 seconds
- **Source:** https://soundbible.com/tags-turkey-gobble.html
- **License:** Free with attribution
- **Usage:** Plays randomly every 2 seconds (5% chance)

### Gunshot Sound
- **File:** `gunshot.mp3`
- **Duration:** ~0.5 seconds
- **Source:** https://soundbible.com (search "rifle shot" or "gunshot")
- **License:** Free with attribution
- **Usage:** Plays when turkey is clicked

### Optional: Hit Sound
- **File:** `hit.mp3`
- **Duration:** ~0.3 seconds
- **Source:** https://soundbible.com (search "impact" or "hit")
- **License:** Free with attribution
- **Usage:** Plays after gunshot when turkey is hit

## Notes:
- All sounds should be MP3 format for broad browser compatibility
- Keep file sizes small (<100KB each)
- Normalize volume levels for consistent playback
- Test across browsers (some require user interaction before playing audio)
