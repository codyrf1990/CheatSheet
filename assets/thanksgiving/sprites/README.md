# Thanksgiving Sprites

## Required Assets:

### Turkey Sprite Sheet
- **File:** `turkey-sprite.png`
- **Dimensions:** 80×100px per frame
- **Layout:** Horizontal sprite sheet
  - Frames 1-4: Walking animation (4-frame walk cycle)
  - Frames 5-6: Death animation (2 frames)
- **Total size:** 480×100px (6 frames)
- **Source:** https://opengameart.org/content/4-frame-walk-cycles
- **License:** Open source / CC0

### Autumn Leaf Images
- **Files:** `leaf-1.png`, `leaf-2.png`, `leaf-3.png`, `leaf-4.png`, `leaf-5.png`
- **Dimensions:** 30-60px each (varied sizes)
- **Colors:**
  - leaf-1.png: Harvest Orange (#FF7518)
  - leaf-2.png: Dark Red (#C94B4B)
  - leaf-3.png: Gold (#D4AF37)
  - leaf-4.png: Autumn Brown (#8B4513)
  - leaf-5.png: Burnt Orange (#FEAC5E)
- **Format:** PNG with transparency
- **Source:** Create from free SVGs or download from OpenGameArt.org

## Notes:
- Turkey sprite should face right by default (walking direction)
- Leaf images should have varied shapes for natural effect
- All images should be optimized for web (compressed PNG)
