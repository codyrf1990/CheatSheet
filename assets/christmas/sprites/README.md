# Christmas Sprites

Place the following sprite files here:

## Gingerbread Man (Required)
Download from: https://opengameart.org/content/gingerbread-man-0

After extracting, you need these walk frames:
- `gingerbread_walk_1.png`
- `gingerbread_walk_2.png`
- `gingerbread_walk_3.png`
- `gingerbread_walk_4.png`
- `gingerbread_walk_5.png`
- `gingerbread_walk_6.png`

**Specs:** 48x64 pixels per frame, will be scaled up in-game

## Present (Required)
- `present.png` - Gift box that appears when gingerbread is hit

**Specs:** ~60-80px, transparent PNG

## Snowball (Optional)
- `snowball.png` - Thrown projectile
- `snowball_splash.png` - Impact effect

**Specs:** ~20-30px, white circle with slight blur/glow
