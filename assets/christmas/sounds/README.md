# Christmas Sounds

Place the following sound files here:

## Game Sounds
- `snowball_throw.mp3` - When clicking/throwing
- `snowball_hit.mp3` - When hitting a gingerbread
- `gift_wrap.mp3` - When present appears (optional)
- `jingle.mp3` - Streak bonus sound (optional)

## Ambient (Optional)
- `fireplace_ambient.mp3` - Crackling fire background

**Format:** MP3 or WAV, keep files small for web performance
