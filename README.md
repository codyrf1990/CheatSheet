# SolidCAM Packages & Maintenance Cheat Sheet

## Overview
A web app for quickly referencing SolidCAM package details, maintenance, and calculator tools. Built with HTML, CSS, and JavaScript. 

## Features
- Package and maintenance cheat sheet
- Calculator for quick math
- Copy-to-clipboard functionality
- Drag-and-drop support
- Persistent data storage

## Getting Started
1. Clone the repo:
   ```bash
   git clone https://github.com/codyrf1990/CheatSheet.git
   ```
2. Open `index.html` in your browser.

## Usage
- Use the calculator for quick math operations.
- Reference package and maintenance info.
- Drag and drop items as needed.
- Copy info with one click.

## Customization
- Update images in `assets/img/` for branding.
- Edit styles in `assets/css/main.css`.
- Extend functionality in `assets/js/`.
