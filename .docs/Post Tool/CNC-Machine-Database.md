## SKU Classification Summary

### Post-3X ($1,500) - 3-Axis Milling
Basic vertical/horizontal machining centers with XYZ linear axes only

### Post-4X ($2,000) - 4-Axis Milling
Machines with 3 linear + 1 rotary axis (typically HMC with B-axis or VMC with 4th axis indexer)

### Post-5X ($2,500) - 5-Axis Milling
True 5-axis simultaneous machining (trunnion table, head-table, or head-head configurations)

### Post-Turn ($1,000) - Basic Turning
2-axis lathes (X/Z only) without live tooling or C-axis

### Post-MT1 ($2,000) - Mill-Turn 1 Channel
Single channel lathes with C-axis and/or live tooling and/or Y-axis milling capability

### Post-MT2 ($4,000) - Mill-Turn 2 Channel
Dual channel machines with main/sub spindles, twin spindle lathes, synchronous transfer

### Post-MT3 ($6,000) - Mill-Turn 3+ Channel
3+ channel machines, twin turret twin spindle (TTTS), complex multi-channel configurations

### Post-Swiss ($2,500) - Swiss Basic
Basic swiss-type lathes with sliding headstock, single channel

### Post-Swiss-Adv ($4,500) - Swiss Advanced
Advanced swiss with B-axis, complex tooling, dual channel capability

### Post-Swiss-3Ch ($7,000) - Swiss 3+ Channel
Multi-spindle swiss machines (6-spindle, 8-spindle), 3+ channel configurations
